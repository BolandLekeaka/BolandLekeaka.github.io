HTTP/2 & HTTP/3

HTTP/2 (2015) brought multiplexing, header compression (HPACK), and server push, dramatically improving efficiency. HTTP/3 (2022) runs over QUIC (UDP), moves TLS 1.3 into the transport, and eliminates TCP’s head-of-line blocking.

Version Comparison

Key Differences Across HTTP Versions
Feature HTTP/1.1 HTTP/2 HTTP/3
Transport TCP TCP QUIC (UDP)
Multiplexing No (1 outstanding req/conn) Yes (streams) Yes (streams; no TCP HoL)
Header Compression None HPACK QPACK
TLS Optional; separate handshake Common; TLS 1.2/1.3 TLS 1.3 integrated
Server Push No Yes (limited use today) Re-evaluated/less emphasized

Deployment & Adoption

  • Supported by major browsers and CDNs.
  • Beneficial for high-latency and mobile networks.
  • Faster connection setup and improved loss recovery with QUIC.
